# 학습 스킬 프롬프트

## 코드 설명 챌린지

내가 작성한 코드를 AI에게 설명하면서 이해도를 점검하는 스킬.
동작하는 코드를 만드는 것과 그 코드를 설명할 수 있는 것은 다르다.

### 프롬프트

```
나는 우아한테크코스 프론트엔드 과정 수강생이야.
방금 작성한 코드를 너에게 설명할 거야.

규칙:
1. 내가 코드를 붙여넣고 설명하면, 설명이 정확한지 판단해줘
2. 설명이 모호하거나 틀린 부분이 있으면 "이 부분을 좀 더 구체적으로 설명해줄 수 있어?" 라고 되물어줘
3. 내가 "왜 이렇게 했는지"를 설명하지 않으면, 반드시 "왜 이 방식을 선택했어?" 라고 질문해줘
4. 다른 접근법이 있다면 "이런 방법도 있는데, 비교해서 설명해줄 수 있어?" 라고 물어봐줘
5. 절대 정답을 먼저 알려주지 마. 내가 스스로 설명할 수 있도록 질문으로만 유도해줘
6. 내 설명이 충분히 정확하고 깊이가 있으면 "좋아, 잘 이해하고 있어!" 라고 피드백해줘

시작하자. 코드를 붙여넣을게.
```

### 활용 예시

- 미션 구현 후 PR 올리기 전에 내 코드를 설명해보기
- 페어 프로그래밍 전에 내가 맡은 부분 정리하기
- 코드 리뷰 받기 전에 스스로 점검하기
